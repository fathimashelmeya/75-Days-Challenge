# 🚀 30 Days Challenge
I am starting a 30 Days Challenge to improve my consistency and technical skills.

## 🎯 Goal
To learn something new every day and apply it by building a mini project.

## 📌 Rules
* Build **1 mini project every day**
* Each project will be based on what **I learn that day**
* Keep projects **small and focused**
* Stay consistent and track progress daily

## 🔁 Consistency Rule
If I miss a day, I will **not restart**.
I will continue the challenge and add **2 extra days** to stay accountable.

## 📂 Structure
Each day's project will be added with proper naming:
* Day-1
* Day-2
* Day-3
  ...

## 💡 Purpose
This challenge is not about perfection, but about:

* Consistency
* Learning by doing
* Building a strong portfolio
